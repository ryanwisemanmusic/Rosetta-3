# Rosetta 3

Apple dropping Rosetta 2 is why I am writing Rosetta 3. Losing access to this software means many developers need some alternative, however, better than where Apple left the software.

Rosetta 3 is intended to not only translate x86/x64/DOS to ARM64, but also tackle win32 related code. Since Rosetta 2 lacks many of this functionality, Rosetta 3 aims to fix what was never delivered.

General features:
- ABI Handshake Validation Layer
- FASM, NASM, MASM Assembler
- Windows Binary Handler

Files:
- .bat file processing
- .dll file processing
- .exe file processing
- .ps1 file processing
